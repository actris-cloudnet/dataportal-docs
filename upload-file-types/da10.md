# Vaisala DIAL DA10

Vaisala DIAL Atmospheric Profiler DA10.

Instrument ID: `da10`

## Recommended files

| File            | Format | Description                                |
| --------------- | ------ | ------------------------------------------ |
| `DA10_ABS_*.nc` | netCDF | Attenuated backscatter files.              |
| `DA10_WV_*.nc`  | netCDF | Water vapor mixing ratio files (optional). |

## Example

Example configuration for [`cloudnet-submit`](https://github.com/actris-cloudnet/cloudnet-submit):

```toml
[[instrument]]
site           = "lindenberg"
instrument     = "da10"
instrument_pid = "https://hdl.handle.net/21.12132/3.9c7bcece918642ea"
path_fmt       = "/data/da10/%Y%m%d/DA10_*_V4610942_%Y%m%d*.nc"
```
